# Backgrounds

Place Figma-exported template backgrounds here.

## Naming Convention
- `SC-Letterhead-P1.png` — Page 1 (full header with wordmark)
- `SC-Letterhead-Cont.png` — Continuation pages (minimal running header)

## Export Specs
- Format: PNG at 300 DPI (2550 x 3300 px) for US Letter
- Alternative: PDF (vector) for print-quality output
