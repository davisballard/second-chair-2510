# New Unit Briefs — 8 Net-New Units (Format Diversity Additions)

*Production-ready briefs for the 8 net-new units Davis greenlit during the 2026-04-23 agency creative review. Each tests a specific hypothesis Indigo named. Each ships with the Saturday 2026-04-25 production batch. Companion to `Agency_Creative_Review.md`.*

**Format mix breakdown (Davis greenlit):**
- 4 Motion Still units (Format A — single image + subtle motion, ~5-6s each) — one per persona
- 2 direct-response variants (cinematic 15s with direct-ASK copy)
- 2 designed static units (Hero Number + Exhibit)

**Total net-new: 8 units, bringing package to 32 total** (24 existing + 8 net-new).
**Total additional production time:** ~3-4 hours on top of existing 24-hour Sat–Mon window.
**Total additional production cost:** ~$8-12 (rounding error against media spend).

**Three testing hypotheses:**

1. **Motion Still beats cinematic for sound-off comprehension** (4 MS units test this — one per persona)
2. **Direct-response beats empathy for high-intent non-soft-tissue audience** (2 DR units test this — S1 and M1)
3. **Designed static carries the brand on Feed** (2 designed-static units test this — HN-1.7B and EX-Settlement-Gap)
