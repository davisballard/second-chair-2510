## What this folder is

The agency campaign book — a navigable hub of one-page briefs that summarize and link to the deep operational docs. Built so Davis can scan the whole pilot in 10 minutes, shift levers in any single brief, and use these as the artifacts to walk Robert through.

**The deep operational docs live elsewhere** — `../Audience/`, `../Creative/`, `../In_Depth/`, `../_Research/`. This folder is the synthesis layer above them. Edit a brief here when the strategic frame shifts; edit the deep doc when the underlying number / spec / persona changes.

## What to read first

- **Just got dropped in?** Start with **01_Campaign_Brief.md** (5 min) → then 03_Media_Mix.md and 04_Creative_Slate.md (10 min combined). You'll have the whole picture.
- **Walking Robert through?** Open **01_Campaign_Brief.md** as the deck cover, then 02_Strategy_Brief.md for the why-it-works story, then 04_Creative_Slate.md for the work itself. Skip 03 for Robert (operational).
- **Shifting media/budget levers?** Open **03_Media_Mix.md**, change the row, then update the deep doc at `../Audience/06_Media_Plan_And_Budget_Allocation.md` to match.
- **Updating the asset list?** Open **04_Creative_Slate.md** + the live Anchor Template Spec at `../Creative/Anchor_Template_Spec.md`.
- **Questioning a past decision?** Open **05_Decision_Log.md** for the rationale + dates + who flagged it.
