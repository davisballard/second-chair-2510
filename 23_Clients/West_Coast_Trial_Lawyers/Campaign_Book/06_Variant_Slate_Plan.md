## Day 7 + Day 14 read framework

### Day 7 (end Week 1, 2026-05-13)

**Only readable signal at this volume = hook rate.** No cell-level lead CPL is statistically meaningful by Day 7.

| Metric | Threshold | Action |
|---|---|---|
| Hook rate (3-sec view / impressions) | >25% healthy | <20% pause that hook's variants |
| CTR | >0.8% after 1K imps | <0.8% pause that unit |
| Anchor cohort vs Clean (FB Feed) | Anchored ≤ Clean CPL | Validate per existing Phase 1 split-test plan |
| Anchor cohort vs Clean (IG Reels) | Clean ≤ Anchored CPL | Validate |
| Motorcycle impression share | >15% | <15% triggers two-ad-set hot-swap |

**Decision rule Day 7:** *kill underperforming hooks, do NOT add new creatives.* Let learning consolidate on the survivors.

### Day 5 early-warning + Anchor hook-rate gate

**Anchor hook-rate gate (revised 2026-05-01):** at Day 5, compare each Anchor variant's 3-second view rate against its matched Clean primary (same footage, no panel). Decision rule:
- **Anchor hook rate within 15% of Clean** (or higher) → GO. Build remaining ~9 Anchor composites in Week 2 production cycle.
- **Anchor hook rate drops >15% with no CPL offset** → KILL the Anchor pattern entirely. Do not build out further. Pause the 4 launched Anchor variants; freed impression share goes to Clean cohort.
- **Mixed (within 15% on some pairs, >15% drop on others)** → diagnose by copy register. Variant A (Promise) underperforming might mean Anchor hurts mid-cinematic-content register; Variant C (Speed) underperforming might mean question-format hooks need Clean breathing room. Decision: keep winning Anchor pairs, kill losing ones, do not scale.

**Simon's two-ad-set hot-swap (separate trigger):** if by Day 5 the Anchor cohort has taken >70% impression share, **pull two-ad-set fallback before the Day 7 gate.** Carve $1,500 for Diana / Jason cluster. Don't wait for the formal read.

(Both Day 5 triggers can fire independently; they're measuring different things — hook-rate diagnostic vs. impression-share imbalance.)
