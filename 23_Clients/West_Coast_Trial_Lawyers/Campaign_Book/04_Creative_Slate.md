## Slate Architecture

Six creative concepts × multiple format variants per concept = ~32-45 total units. Each unit serves a different format-mood-state-persona combination.

**Aspect-ratio convention for this slate:**
- Primary aspect listed is the NATIVE generation ratio
- "+" notation means additional crops/adaptations exist or should be built (e.g., `9:16 + 4:5` means native vertical + Feed-portrait crops)
- **1:1 (square) crops are deprecated as of 2026-05-01** — Davis's call: "I never see them, they don't work anymore." Meta's own data also shows 4:5 = +1% CTR vs 1:1 and 9:16 = +7% CTR vs 1:1. All builds are 9:16 + 4:5 only.

**Script column convention:**
- Video units: `Hook` (the opening line/headline) + brief beat description; full scripts in `../Creative/Week_1_Approval_Package/` per unit
- Static units: `n/a` (no script — visual + caption only)
- TBD: Davis fills in for variants without locked scripts yet

## Aspect-Ratio Production Map

| Surface | Aspect | What renders here |
|---|---|---|
| FB Feed | 4:5 (portrait) | All cinematic + Anchor + Designed Statics + DR + Static Laura. Meta data: 4:5 outperforms 1:1 and is the dominant Feed format. |
| FB Reels + IG Reels + IG Stories | 9:16 (vertical) | All video native; Motion-Still; Anchored variants render Anchor at full bottom 35-40% |
| IG Feed | 4:5 | Same as FB Feed |

**Operational rule:** every unit has a 9:16 native + 4:5 crop. **1:1 deprecated 2026-05-01** — Davis: "I never see them." Meta's data confirms: 4:5 = +1% CTR vs 1:1; 9:16 = +7% CTR vs 1:1. Static units get pure JPG for FB Feed surfaces and 4-7s ambient-motion MP4 for IG Reels/Stories surfaces (per surface-aware static-vs-video rule in `03_Media_Mix.md`).
