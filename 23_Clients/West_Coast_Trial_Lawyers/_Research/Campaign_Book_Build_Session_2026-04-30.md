## Decisions

1. **Folder structure:** `Campaign_Book/` at WCTL client root with 5 numbered briefs + README index (00-05)
2. **Brief lengths:** 1-2 pages max each; substance-deeper-than-2-pages belongs in operational source-of-truth docs (link, don't reproduce)
3. **Comms Plan format:** dual — 9-grid spine + Living/Looking/Buying journey overlay
4. **Media Mix format:** one-page matrix
5. **Creative Slate format:** asset-manifest table with sortable columns
6. **Decision Log:** chronological + open-items + locked-decisions sections
7. **Davis-edits convention:** numbered prefixes stable; new briefs use next number (06_, 07_, etc.); index updated on add
8. **Source-of-truth links:** every brief links to the deep doc rather than reproducing the deep substance
