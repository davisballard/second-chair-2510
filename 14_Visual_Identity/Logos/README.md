# Second Chair Logo Files

## Folder Structure

```
Logos/
├── Gold/          ← Gold variant (primary for oxblood backgrounds)
├── Oxblood/       ← Oxblood/burgundy variant (for cream backgrounds)  
├── Black/         ← Single-color dark (print, engraving, embossing)
└── White/         ← Single-color light (reversed print, white surfaces)
```

## File Naming Convention

```
second-chair-[component]-[color-variant].[ext]
```

| Component | Description |
|---|---|
| `logomark` | The mark only — two columns in concentric circles |
| `wordmark` | The typographic mark only — SECOND / rule / Chair |
| `lockup` | Combined — logomark left + wordmark right (horizontal) |
| `lockup-stacked` | Combined — logomark above + wordmark below |

| Name | Hex | Usage |
|---|---|---|
| Old Gold / Brass | `#E8C165` | Gold variant — mark and wordmark on oxblood |
| Burgundy / Oxblood | `#490A0A` | Oxblood variant — mark and wordmark on cream |

| Cream / Ivory | `#FFF7F0` | White variant — reversed, light applications |

## Wordmark Spec (for reference)

- **"SECOND"**: Tiempos Headline Bold (Klim Type Foundry)
- **Rule**: 0.5pt horizontal line, exactly the width of "SECOND"
- **"Chair"**: Adobe Caslon Pro Regular, ~45% the point size of SECOND, +10–15 tracking

## Logomark Spec (for reference)

- Two Doric columns, side by side, gap equal to one shaft width
- Two concentric circles: inner fine rule, outer ~1.5x heavier
- Vectorized from AI-generated flat black linework, cleaned in Illustrator
