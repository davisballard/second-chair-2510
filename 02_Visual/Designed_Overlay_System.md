# Second Chair — Designed Overlay System

*The non-footage visual layer for SC ads. Text, graphics, charts, hero numbers, exhibits, lower thirds, transitions. Static by default, motion when it adds meaning. The bar: highly designed, evokes the right register, every overlay earns its place.*

**Framework owner:** the standing SC ad production room (Giuseppe + Massimo + Graham Fink + Ed McCabe + Indigo Sato).
**Established:** 2026-04-23, Abracadabra agency bring-in (Nigel / Hegarty / Massimo / McCabe / Indigo).
**Pressure-tested by:** Hegarty / Indigo / McCabe / Massimo against the conversion goal before commit.

---

## §0 — THE PRINCIPLE (read this first)

The framework has two parts, in order. Both are non-negotiable.

### Part 1 — Overlays Earn Their Place

Overlays are a tool used when they make the spot better. **Not because we built a system.** If an overlay doesn't serve comprehension, credibility, or conversion for *this specific spot*, it doesn't ship. Most SC spots will use one or two overlay moves; some will use none at all.

**The spot is the unit, not the framework.** Don't apply patterns by default — apply them when they earn their place.

### Part 2 — Design-First, Motion-When-It-Serves

When an overlay IS used, the bar is non-negotiable: **highly designed, evokes the right register** (prestige, rigor, restraint, trust, specificity).

Most can be static. A beautifully composed static overlay does the comprehension/credibility work as well as a kinetic one — often better, because there's nothing to wait for. **Motion is one tool — used when it adds meaning** (narrative tension, reveal-as-punchline, structural transition between scenes), never as decoration.

The static-or-motion test: *if motion doesn't add narrative meaning, the static version is the right answer.*

### The Bar

- **What SC refuses:** white-box-bad-font overlays. Templated stat blocks. Generic kinetic captions in a stock sans-serif. CTA slabs.
- **What SC sets:** every overlay that ships reads as a designed object that belongs in the same world as the footage.

### The Conversion Goal That Anchors All of This

**Qualified case fills from people who have actually been in an accident.** Not raw clicks. Not impressions. Not "engagement."

The framework's restraint and specificity disciplines exist *because* they self-select for the qualified lead. Disciplined ads convert the right people well; over-decorated ads convert everyone badly. Specificity in the hero number filters in people who've been through the territory; restraint in design signals "real firm, not scam"; brevity in copy filters out the unqualified.

### Agency Pressure-Test (2026-04-23)

The framework's discipline rules — Indigo's >30% hook-rate gate, McCabe's brevity-as-filter, Massimo's "two design moves max per frame" — collectively make over-decoration impossible by construction. The framework was validated against the conversion goal before being committed. This isn't a beautiful artifact; it's a tool that survives the scroll.

## §1 — WHAT WE EVOKE

The five registers SC's design system carries. Every overlay decision should be checkable against this list: *what is this evoking? Is it the right register for this spot?*

| Register | What it is | How to render it |
|----------|-----------|------------------|
| **Prestige** | The "real firm with real lawyers" register | Tiempos Headline + Caslon body + cream background (never white). Type set with editorial weight. |
| **Rigor** | The "this is analytical, not promotional" register | EXHIBIT N: naming convention. McKinsey-grade conclusion-as-title. Tufte data-ink discipline. Source attribution. |
| **Restraint** | The "the only adult in the category" register | Single accent color (burgundy only). Generous negative space. Two design moves max per frame. What's removed makes the work. |
| **Trust** | The "you can rely on this" register | Optical small caps (true, not faux). Warm charcoal not pure black. Hierarchy through typography, not color. Old-style figures in body. |
| **Specificity** | The "this is for me, this is real" register | The one number, the dated source, the named exhibit. Never vague claims. "30%" not "much less"; "$150,000" not "significant"; "EXHIBIT 3:" not "see this chart." |

Most spots will hit two or three of these registers. The Hero Number drop = Specificity + Restraint. The Exhibit reveal = Rigor + Prestige. A static title card = Prestige + Restraint. **Identify which two-three registers the spot needs, then design to them.**

## §2 — WHEN TO REACH FOR AN OVERLAY (and when not to)

Decision check before adding any overlay element to a spot:

| Question | If YES | If NO |
|----------|--------|-------|
| Does this overlay help the scroller comprehend what the ad is about within 1.5s? | Add it | Don't add it for comprehension reasons |
| Does this overlay carry a credibility signal the footage alone doesn't carry? | Add it | Don't add it for credibility reasons |
| Does this overlay carry the *one* most important number/word/claim of the spot? | Add it | Don't add it for emphasis reasons |
| Is this overlay required to land the CTA at the close? | Add it | Use diegetic finish + Meta button instead |
| Would the spot work without this overlay? | If YES → strongly consider removing | — |

**Default state for every spot starts at zero overlays.** Add one when it earns its place. Add a second only if it earns its place separately. Three overlays in a single 10-20s spot is the maximum; most great spots will have one or two.

**The CTA frame is not an overlay** — it's the closing diegetic finish + Meta button per the SC short-form rule. Don't double-overlay the CTA with kinetic text on top of the closing frame.

## §3 — KINETIC TEXT CADENCE (existing SC rule, integrated)

This is the SC creative spec that already governs kinetic captions across every ad. Owned by the Ad Creative Roundtable (`../../../02_Creative/_Roundtable/SKILL.md`). Codified here as a reference because it interacts with §6 patterns.

- **5-6 phrases at 2-3s each.** Continuous throughout the ad — not big static cards.
- **Bottom-center**, with 250px safe-zone padding above the bottom UI controls.
- **No cold-open CTA slabs. No mid-roll text-on-black CTA cards. No stinger-CTA blocks.** These break narrative in 10-20s format.
- **Boardroom pattern is the default:** continuous kinetic text + diegetic finish + Meta button does the closing work.

The kinetic captions live in the bottom-center safe zone. The §6 signature patterns (Exhibit / Hero Number / Burgundy Rule) live in the upper-middle 60% of the frame. They coexist but don't compete spatially.

## §4 — THE FIVE DESIGN PRINCIPLES (apply to static AND motion)

### Massimo's Typography Principles

**1. Hierarchy through order and weight, not animation gimmicks.**
The eye reads the same hierarchy on a still page that it reads in motion. EXHIBIT label first (smallest, top, Caslon small caps, Warm Grey). Headline second (Tiempos, Warm Charcoal). Data third (in burgundy when present). CTA last. Each element holds its place by typographic weight, not by motion fanfare.

**2. Optical small caps don't pop, they settle.**
Static execution: the type appears weighted-correctly in its final state, no "popping." Motion execution: the type fades up at 60% opacity, then reaches 100% with a 0.3s tracking-tighten as it locks. Like ink drying on paper. The amateur version pops the type in with a bounce; the elite version brings it home like it was always there.

**3. The Burgundy Rule is structural, not decorative.**
Static execution: placed deliberately as a divider; one or zero per frame; never decorative. Motion execution: draws across the frame from left to right at a measured 0.4-0.6s — same pace as the eye reading the headline above it. One draw, one lock, no re-animation. If a spot uses the Burgundy Rule more than once, each instance is structural (between scenes), not repeated for visual rhythm.

### Bass-Perspective Design + Motion Principles

(Channeled from `../../../04_Production/Giuseppe_Karma_(AI_Creative_Director)/Designed_Overlay_Reference/Saul_Bass.md`.)

**4. Designed-IN via material reference.**
Even static graphics share light/grain/perspective with the footage they overlay. SC's color/material origins are documented (West Federal Reporter spine, Harvard Law Review paper, brass door placards). The graphics inherit that material logic. Bars don't "animate in" — they **fill in like ink on cotton-rag paper.** Numbers don't "count up" — they **resolve from blur to sharp**, like a focusing camera. Static graphics feel printed, not pasted. This is the difference between a designed-in element and a separate-plane overlay.

**5. Two beats, not five (when motion is used).**
This is the discipline that separates SC from generic motion graphics. Most brands animate every chart element separately — bars grow, then labels appear, then the title flies in, then the source attribution slides up, and by then the user is gone. SC convention: **EXHIBIT label appears (beat 1) → entire chart resolves to final state (beat 2).** Two beats. Total duration 1.0-1.5s for the chart to land. Then it holds. The reader gets to *read* the chart, not watch it perform.

### The Static-or-Motion Decision Test

Before any pattern is set in motion, the production room asks: **does motion add narrative meaning here?**

- Reveal-as-punchline (the chart appears at the moment of greatest narrative tension) → motion earns it
- Structural transition (the burgundy rule between two scenes) → motion earns it
- Element draws attention to itself for no narrative reason → motion does NOT earn it; render static

The static variant is the default. Motion is the exception.

## §5 — THE 4-JOB TIME BUDGET

Every SC spot, regardless of format (Motion Still / Story Slide / Luca Maxim Cinematic), should be readable as this beat structure:

| Time | Job | Mechanism |
|------|-----|-----------|
| **0.0 – 0.5s** | Comprehension lock | EXHIBIT label appears top-left in Caslon small caps + Tiempos headline (3-5 words: "Hurt in a wreck?" / "What insurers won't tell you" / "$0 upfront") — usually static |
| **0.5 – 1.5s** | Credibility signal | Burgundy rule placed (or drawn); cream background already establishes register; the prestige type has done its job before any data appears |
| **1.5 – 8s** | The case | 5-6 phrase kinetic cadence per existing SC rule (bottom-center captions); hero number drops at the highest-tension beat (static or kinetic depending on the spot); Exhibit chart appears at the moment the VO/kinetic text sets up the question |
| **8 – 12s** | The call to act | Single CTA in Tiempos: "Get your case reviewed." / "See what your case is worth." Restraint. No CTA slab. Diegetic finish + Meta button per existing SC short-form rule |

**Test for whether a spot is following the framework:** can it be unbundled into this beat structure? If yes, it follows the framework. If no, it's improvising. Improvisation is fine for hero spots; not for the 5-10 ads/week iteration pace.

## §6 — THE FIVE SC SIGNATURE PATTERNS

Each pattern has a default static form (used most of the time) and a motion variant (used when motion adds meaning).

Patterns 1–3 are non-footage compositions (rendered as Higgsfield stills or CapCut shapes, often used in designed-static units or chroma-keyed onto cinematic frames). Patterns 4–5 (added 2026-04-25) are CapCut-native text layers built to sit *over* generated footage — they own the sound-off scroll-stop and VO-emphasis problems on Reels.

### Pattern 1 — The Exhibit

**Function:** data-viz authority moment. Used when the ad's claim is best supported by a chart, comparison, or visual statistic. The Exhibit label is the headline; the chart is the proof.

**Reference lineage:** McKinsey + Tufte + IBM under Paul Rand + (motion variant) Patrick Clair living-photograph technique.

**Default — static:**
- Cream background `#FFF7F0`
- "EXHIBIT N:" in Adobe Caslon Pro, true optical small caps, 9pt, Warm Grey `#6E6862`, top-left
- Headline in Tiempos Headline, 36-48pt, Warm Charcoal `#1C1917`, beneath the EXHIBIT label
- Single horizontal bar chart below the headline: one short grey bar (Warm Grey), one long burgundy bar (Burgundy `#490A0A`)
- Labels in IBM Plex Mono 8pt, right-aligned numbers
- One thin 0.5pt burgundy rule separating the EXHIBIT label section from the chart
- Generous negative space
- 9:16 vertical, full frame

**Motion variant — Exhibit Reveal (1.5s, chroma-key path):**
1. EXHIBIT label fades up at t=0.0–0.3s (Massimo's "settling" motion)
2. Burgundy rule draws left-to-right across t=0.3–0.7s
3. Chart bars resolve to final state at t=0.7–1.5s (Bass two-beat principle, Patrick Clair living-photograph quality — bars don't slide, they *resolve* into final form)
4. Holds for the rest of the spot beat

**Use the motion variant when:** the chart appears at the moment of greatest narrative tension, after the VO or kinetic text has set up the question the chart answers. Otherwise default to static.

### Pattern 2 — The Hero Number

**Function:** single hero stat lands. Used when the spot has ONE number that carries the entire claim. Candidates for SC: $150,000 (settlement gap), 30% (insurance offers less to unrepresented), $0 (upfront cost), 1 in 3 (statistic anchor).

**Reference lineage:** Apple keynote comparison reveals + Bass distillation + Public/Robinhood/Cash App in-feed data-viz.

**Default — static:**
- Pure black `#000000` background (for blend-mode overlay) OR cream `#FFF7F0` background (for direct frame use)
- Single number in Tiempos Headline, 120-180pt, white (on black) or Warm Charcoal (on cream)
- Generous tracking, optical small caps style
- No other elements — no label, no source, no decoration
- The number IS the frame

**Motion variant — Number Drop (0.8s, blend-mode path):**
1. Number resolves from 8% blur to sharp over t=0.0–0.6s (Patrick Clair living-photograph technique)
2. Tracking tightens 5% as the number sharpens (settles into final state)
3. Holds for 1.5s minimum after resolve

**Use the motion variant when:** the narrative tension calls for the resolve — when the spot has built up to the number and the reveal IS the punchline. For comparison numbers ($2,500 vs. $150,000), the second number can use the motion drop while the first is static, signaling which is the point.

### Pattern 3 — The Burgundy Rule

**Function:** structural divider. Used when the spot needs to signal a transition between scenes, sections, or beats. The rule is structure, not decoration.

**Reference lineage:** West Federal Reporter spine + Vignelli grid discipline + (motion variant) Bass title-sequence rule-draws.

**Default — static:**
- Thin 0.5pt burgundy `#490A0A` horizontal rule
- Placed deliberately as a divider on the frame
- Full width or 80% width depending on composition
- One rule per frame max — never two

**Motion variant — Burgundy Rule Draw (0.5s, blend-mode path):**
1. Rule draws left-to-right across the frame at t=0.0–0.4s (Massimo's "draws once and locks")
2. Holds for the rest of the beat
3. Never re-animates

**Use the motion variant when:** the rule is functioning as a transition between two scenes. For static placement within a single frame (e.g., separating EXHIBIT label from chart in The Exhibit pattern), the rule appears in its final state — no separate animation.

### Pattern 4 — The Hook Card

**Function:** the 0–2s scroll-stopper. Sound-off comprehension lock for Reels-feed viewers. Signals "this is for me, this is real, this is from a credible source" before the VO opens. Used at the top of nearly every spot where the visual register on its own doesn't carry the territory in 0.5s.

**Reference lineage:** Bass title-card discipline (graphic distillation, the type IS the image) + New Yorker / Pentagram editorial cover logic + A24 trailer typography. Per-register variants — Editorial vs UGC-Native — share the underlying restraint but render to different vocabulary.

**Palette note:** Hook Card uses the SC parent overlay palette (Cream / Warm Charcoal / Burgundy), NOT the Fair Case brand palette (Roman Navy / Torch Amber). Fair Case's palette is built for controlled-background contexts (landing page, form, static) and fails as letter-fill over warm-graded video footage. Fair Case branding lives at the final CTA frame only. See `../../20_Fair_Case/04_Visual_Identity_Brief/BRAND_SYSTEM_BRIEF.md` Ad Creative Overlay Rule.

#### Editorial register (Elevated Documentary spots — S1, M1, M2 primaries)

**Default — static:**
- Headline: Tiempos Headline 64–88pt, two lines max, McCabe-tight (3–5 words per line)
- Letter-fill: Cream `#FFF7F0` over warm/dark footage; Warm Charcoal `#1C1917` over bright/cool footage. Massimo picks per footage grade.
- Sub-line (usually used): Söhne Mono 11pt small caps, letterspacing ~80, e.g., `S1 — TRUCKING / I-17 PHOENIX`. Sits above headline. Sound-off context lock.
- Burgundy `#490A0A` 0.5pt rule between sub-line and headline (Bass title-sequence move, structural).
- One contrast device — pick exactly one, never stack:
  - **60% Cream scrim**, gradient fade-down in upper third. Default for most footage.
  - **80% Warm Charcoal scrim**, gradient fade-down in upper third. Used over bright/blown footage (golden-hour gardens, white-room MVA, etc.).
  - **Knockout Burgundy chip** behind a single accent word. At most one Hook Card per ad set uses this — the signature peak hook only.
  - **1px Warm Charcoal outline + 8% directional drop-shadow.** Last-resort device when no scrim works (motivated practical light makes the upper third busy).
- Position: headline anchored at 35–48% Y, sub-line at 28% Y, rule at 33% Y. Composition leaves bottom half clear for footage and captions.
- Hold: 1.5s minimum, 3.0s maximum. Headline retires before VO peaks.

**Motion variant — Bass two-beat reveal (CapCut-native):**
1. Sub-line + Burgundy rule fade up at t=0.0–0.3s (Massimo's "settling" — opacity 0→100, no scale, no slide).
2. Headline letterforms resolve from 8% blur to sharp at t=0.0–0.4s (Patrick Clair living-photograph technique). Tracking tightens 3% over the resolve.
3. Holds. **No second animation. No pulse. No re-trigger.**

**Use the motion variant when:** the spot opens on a footage hold and the headline needs to *appear* rather than be present from frame 1. For most Reels, where the user has just scrolled into the frame, the static variant from frame 1 is the right answer — every frame the user sees has the headline doing its work.

#### UGC-Native register (S5 Diana, MS-Diana, MS-Laura)

**Default — static:**
- Headline: Inter Black 56–64pt, two lines max, white `#FFFFFF` letter-fill with 2px black `#000000` stroke
- Sub-line (optional): Inter Bold 14pt italic, white with 1px black stroke
- Position: top-third or bottom-third, 5% from frame edge. Avoid center.
- No scrim. No rule. No Burgundy chip. **The black stroke is the device.**
- Register: iPhone-Notes / native phone-text. Reads as a screenshot, not a designed overlay. Patrick Clair would call this *the absence of design as a design choice* — the discipline is to not over-design.
- Hold: 1.5s minimum, 3.0s maximum.

**Motion variant — snap-cut or hard-fade:**
- Snap-cut on (0 frames). Default.
- Hard-fade in 4 frames (~0.13s at 30fps) when snap reads abrupt against a particular cut.
- **No blur-to-sharp. No tracking-tighten. No Bass reveal.** UGC tolerates abruptness; editorial does not. Smuggling editorial motion into UGC produces "glossy UGC" — the failure mode `UGC_Native.md §6.5` calls out.

**Use the motion variant when:** always — UGC Hook Cards either snap on at frame 0 or hard-fade in 4 frames. Pick whichever cuts better against the footage.

#### Time budget for the Hook Card (universal, both registers)

| Time | Job |
|------|-----|
| 0.0–0.5s | Comprehension lock — headline visible and readable |
| 0.5–1.5s | Credibility — sub-line resolved, sound-off viewer locks context |
| 1.5–3.0s | Hold while VO opens |
| 3.0s+ | Headline retires; footage and captions take over |

### Pattern 5 — The Peak Card

**Function:** single mid-spot moment that lands at the VO turn. The graphical anchor for the *one* beat where the sound-off viewer needs to read the spot's claim, not just feel it. Used at the moment of greatest narrative tension — the line the VO is most committed to.

**One Peak Card per spot. Never two.** Massimo + Indigo own the call (which beat earns it). McCabe picks the content (3–8 words). Giuseppe picks the variant.

**Reference lineage:** Patrick Clair living-photograph reveal + Apple keynote comparison-reveals + Bass distillation (a single hero element, weighted restraint).

#### Three variants — Giuseppe picks per spot

**Variant A — Hero Number** (already in Pattern 2; tuned for over-video use)
- Single stat in Tiempos Headline 120–180pt, Cream knockout chip bottom-third or full-bleed cream/black field. See Pattern 2 for production spec (chroma-key / blend-mode pipeline).
- Use when: the spot has ONE number that carries the entire claim. Filters for sound-off viewers who clock the number even if they didn't read the headline.

**Variant B — Defined Term**
- Term: Tiempos Italic 48pt (Editorial) or Inter Black 36pt caps with letterspacing 60 (UGC), Warm Charcoal on Cream knockout chip (Editorial) or white-with-2px-stroke (UGC).
- Definition line: Söhne 14pt regular caps with letterspacing ~50, Warm Grey, beneath the term (Editorial). For UGC: Inter Bold 14pt caps, white with 1px stroke.
- 0.5pt Burgundy rule between term and definition (Editorial only — UGC drops the rule).
- Format: `FMCSA / federal motor carrier safety administration` or `RECORDED STATEMENT / what insurance asks for before you have a lawyer`.
- Use when: the VO uses a legal/industry term whose meaning the sound-off viewer needs anchored in 1.5s. The Peak Card teaches the term while the VO completes the sentence.

**Variant C — Quote Card**
- VO's spoken line as block quote in Tiempos 36–44pt (Editorial) or Inter Black 32pt (UGC).
- Italic for emotional emphasis lines, regular for declarative.
- Burgundy 0.5pt rule below the quote (Editorial only).
- Use when: the emotional peak of the VO is the line itself, and a sound-off viewer needs to *hear it visually*. The line that would be the spot's pull-quote in print.

#### Static or motion (Massimo + Indigo decide)

- **Static default:** Peak Card appears in final state on the cut. Holds 1.5–2.5s. Works for most spots.
- **Motion variant** (Editorial only): element resolves from texture into form via Patrick Clair living-photograph reveal — blur-to-sharp 0.0–0.6s, tracking tightens 3%, no slide, no bounce. Used only when the reveal IS the punchline — when the spot has built up to this moment specifically.
- **UGC motion:** snap-cut or hard-fade. No Patrick Clair reveal in UGC register — that smuggles editorial in.
- Hero Number variant motion: see Pattern 2.

#### Position and timing

- Position: bottom-third typically (above the captions safe zone), or full-bleed for Hero Number.
- Timing: at the VO turn — typically 5.0–8.0s into a 12–15s spot. Some VO scripts frontload the thesis (M1's "They ask for what's fair" lands at 3.5s) — Peak Card lands at the *content* peak, not a fixed time.
- Hold: 1.5s minimum, 2.5s maximum. Long enough to read; short enough to keep momentum.

#### Coexistence with the Hook Card

Hook owns 0–3s; Peak owns the VO-turn moment. They never overlap spatially or temporally. They usually share the same register (both Editorial or both UGC), but a UGC spot can use a Defined Term Peak Card if Indigo decides the educational anchor outweighs register consistency.

### Static template — for any pattern in static use

When using the static variant of any pattern, use only the STILL portion of the relevant template above. Drop the still directly into CapCut as a layered image (no chroma key needed if the still is composed at the correct aspect ratio with the SC palette intact). Use the still for the duration the spot calls for — typically 1-3 seconds for a hero element, structural for the entire spot if it's a Burgundy Rule divider.

### Critical prompting rules for the animation step

The whole game in motion variants is keeping the background dead-still so the key works cleanly. Both Kling and Seedance like to add subtle camera/background drift unless you explicitly forbid it:

- **"static solid color background"** — every prompt, explicit
- **"no camera movement, no parallax, no zoom on background"**
- **"only the [text/graphic] animates"**
- **No "cinematic" or "dramatic" modifiers on the background** — those words trigger background motion
- **Keep durations short (0.5-1.5s for SC patterns)** — fewer frames for the background to drift
- **Patrick Clair "living-photograph technique"** is the key phrase to invoke for resolve-not-slide motion
- **"Saul Bass discipline" / "Bass two-beat reveal"** for the structural restraint

### The Static-or-Motion Decision

Owned by Massimo (system gate) + Indigo (conversion lens). Question they ask before any pattern is set in motion: **does motion add meaning to this specific spot, or is the static version more disciplined here?**

If yes → motion variant ships.
If no → static variant ships.
Giuseppe executes whichever the room calls.

### The Overlay-or-No-Overlay Decision

Default state for every spot is **zero overlays.** The room adds an overlay only when it earns its place per §2. Indigo's veto runs heavy here — if she calls "this overlay doesn't drive qualified-lead conversion," the spot ships without it.

### When to Escalate to Hegarty

The standing room handles the **5-10 ads/week iteration pace** that Indigo's testing requires. Hegarty is NOT pulled in by default — that would make him a bottleneck.

Hegarty is pulled in when:
- The spot is a **hero spot** that will run as the centerpiece of a launch
- The spot **intentionally breaks the framework** (uses an overlay register the framework doesn't cover, or skips a step in the time budget for narrative reasons)
- The spot is **brand-defining** — first time using a new visual approach

For all other spots, the room runs without him.

## §10 — VERIFICATION CHECKLIST (pre-ship)

For any SC ad before it ships, the production room runs this check:

**Overlay justification:**
- [ ] Each overlay in the spot earns its place per §2 (comprehension / credibility / hero number / CTA)
- [ ] The spot is not over-overlaid (max 3 elements, target 1-2)
- [ ] The default state of zero overlays was actively considered and rejected for good reason

**Design discipline:**
- [ ] Each overlay reads as designed (not white-box-bad-font)
- [ ] Each overlay evokes the right register per §1 (prestige / rigor / restraint / trust / specificity)
- [ ] Two design moves max per frame
- [ ] SC's locked typography stack is used (Tiempos / Caslon / Söhne / IBM Plex Mono — no substitutions)
- [ ] SC's locked palette is used (cream + burgundy + warm charcoal — no rogue colors)
- [ ] EXHIBIT naming convention used where data is presented

**Motion decision:**
- [ ] Each motion variant earns its motion (adds narrative meaning, not decoration)
- [ ] Static variants chosen where motion would not add meaning
- [ ] Motion variants follow the two-beat principle (max 1.5s per element to land)

**Conversion check (Indigo's domain):**
- [ ] Spot reads in 1.5s without sound (sound-off literacy)
- [ ] First 1.5s communicates "this is about an MVA" + "why I should care"
- [ ] Single hero number identified and is the right number for this spot
- [ ] Specificity sufficient to filter for qualified leads (people who've actually been in an accident)
- [ ] No CTA slab; closing is diegetic finish + Meta button

**Time budget (Hegarty's framing):**
- [ ] Spot can be unbundled into the 4-job structure (§5)
- [ ] If it can't, the production room has explicitly named why and Hegarty has approved the deviation

If all boxes check, ship. If any don't, iterate.
