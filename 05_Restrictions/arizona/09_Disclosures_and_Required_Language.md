# Disclosures and Required Language (Arizona)

> **Saul:** "Disclosures aren't legal-gobbledygook to bury at the bottom of the page. They're the load-bearing structure that turns a marketing message into ER 7.1-compliant communication. Clear and conspicuous — same size, same color, same contrast as the primary text. AZ takes that standard seriously."

---

## 1. MANDATORY DISCLOSURES (AZ)

| Disclosure | Required? | Placement | Source |
|---|---|---|---|
| Responsible attorney name + contact | :white_check_mark: Yes | In ad or landing page | ER 7.1 |
| "This is an advertisement" | :warning: Best practice for general ads; **mandatory** for direct mail/email | Landing page / direct mail | ER 7.3 (for direct contact); best practice elsewhere |
| "Not a law firm" (for trade names like Fair Case) | :white_check_mark: Best practice | Footer/disclosure section | ER 7.5 |
| TCPA consent language | :white_check_mark: Yes (federal) | Above submit button | FCC 2025 / TCPA |
| Past results disclaimer | :warning: Recommended | Adjacent to any results claim | ER 7.1 (best practice) |
| Actor/dramatization disclosure | :white_check_mark: If applicable | Visible during content | ER 7.1 |
| Contingency fee structure | :white_check_mark: If fees mentioned | With fee information | ER 1.5, ER 7.1 |
| Multi-jurisdictional disclosure | :white_check_mark: If multi-state firm | In ad or landing page | ER 7.5 |

---

## 2. THE "CLEAR AND CONSPICUOUS" STANDARD

ER 7.1 requires disclosures to be "clear and conspicuous" — meaning **same size, color, duration, cadence, contrast, location, and audibility** as the primary content, such that an ordinary person would easily read, hear, notice, and understand them.

| Treatment | Status |
|---|---|
| 4pt font in light gray at the bottom of a black landing page | :x: Not clear and conspicuous — fails ER 7.1 |
| Same body-text size, same color, in-flow with the content | :white_check_mark: Clear and conspicuous |
| Hovering "i" tooltip that requires interaction to see | :warning: Probably not clear and conspicuous; depends on context |
| One-frame disclosure flash on a 15-second video | :x: Not sufficient duration to be readable |
| Disclosure in the same on-screen text register as the primary creative | :white_check_mark: Clear and conspicuous |
| Audio-only disclosure on a video viewed sound-off | :warning: Best practice is to also include on-screen text |

## 3. STANDARD DISCLOSURE COPY BLOCKS (AZ)

### Standard Ad Disclosure (Landing Page)

```
This is an advertisement for legal services. Fair Case is operated by
Second Chair and is not a law firm. By submitting your information, you
consent to being contacted by West Coast Trial Lawyers, a participating
law firm. West Coast Trial Lawyers is responsible for the content of
this advertisement.
```

### Past Results Disclaimer

```
Past results do not guarantee future outcomes. Each case is unique
and depends on its own facts and circumstances. Results may vary.
```

### Multi-Jurisdictional Firm Disclosure

```
West Coast Trial Lawyers is licensed to practice in [list of states
including Arizona]. Cases involving Arizona claimants will be handled
by attorneys licensed in the State of Arizona.
```

### Dramatization / Actor Disclosure

```
Dramatization. Not an actual client. Actors used.
```

OR (more specific):

```
This depiction is a dramatization. Not an actual client testimonial.
```

### TCPA Consent Block (Above Submit Button)

```
By clicking "Submit," I consent to be contacted by West Coast Trial
Lawyers via phone, text, or email at the contact information I
provided, including via automated systems and pre-recorded messages,
regarding my potential legal matter. I understand consent is not a
condition of any service. Message and data rates may apply. I have
read and agree to the Privacy Policy and Terms of Service.
```

(See [`08_TCPA_Arizona_Overlay.md`](08_TCPA_Arizona_Overlay.md) for the full TCPA consent architecture.)

### Direct Mail Solicitation Header (If Ever Used)

```
NOTICE: THIS IS AN ADVERTISEMENT!
ADVERTISING MATERIAL: THIS IS A COMMERCIAL SOLICITATION
```

(Red ink, all caps, double the largest body text size — required on envelope exterior and first page if direct mail to known prospective clients. SC's current operation does not use direct mail; this is reference only.)

## 4. WHAT AZ DOES NOT REQUIRE

| Other-state requirement | Arizona equivalent |
|---|---|
| NY: "Attorney Advertising" label on first page of website | Not required (best practice; including is fine) |
| FL: Spoken "Spokesperson is not an attorney" disclosure | Not required (FL Rule 4-7.22 does not apply) |
| TX: Specific format "Not a law firm" disclaimer | Not explicit AZ requirement; ER 7.5(a) is satisfied with the SC disclosure pattern |
| CA: SB37 crash-footage restrictions | Not applicable (no AZ analogue) |
| CA: Pre-filing of advertisements | Not required |

> **Saul:** "Don't bolt on rules from other states. Each state's disclosure requirements are designed for that state's framework. Putting 'Attorney Advertising' on AZ creative when it's not required just adds visual noise. Match each state's actual rules. We have separate creative variants per state for a reason."

## 6. COMMON DISCLOSURE MISTAKES TO AVOID

| Mistake | Fix |
|---|---|
| Bury "Not a law firm" in 4pt gray | Match body text size and color |
| One-frame disclosure flash on a 15s video | Hold disclosure on-screen for ≥3s; include in audio if voice-over has results claims |
| Settlement amount with no disclaimer adjacent | Pair every dollar amount with the past-results disclaimer |
| Multi-state firm name without licensed-jurisdiction disclosure | Include the list of states where WCTL is licensed |
| TCPA consent block that lists "marketing partners" | Use specific named entity (WCTL) per FCC 2025 |
| No actor disclosure on AI-generated client portrayals | Add "Dramatization" disclosure on every AI-generated client/scene |
