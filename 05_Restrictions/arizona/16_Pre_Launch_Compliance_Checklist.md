# Pre-Launch Compliance Checklist (Arizona)

> **Saul:** "Run this checklist before any AZ campaign goes live. Every box gets checked, every gap gets resolved, every deferral gets documented. If you can't tick a box, the campaign waits."

---

## 1. COPY REVIEW

- [ ] No false or misleading claims (ER 7.1)
- [ ] No unsupported superlatives ("the best," "the most," "#1," "leading," "top-rated")
- [ ] No "certified specialist" language unless WCTL attorneys are AZ Board-certified
- [ ] "Specialize" / "focus on" language is factually supported (post-2021 AZ permits with truth)
- [ ] Past-results disclaimer paired with every results claim
- [ ] Urgency language is factual (e.g., "2-year deadline" cites A.R.S. § 12-542; "180-day deadline for government cases" cites § 12-821.01)
- [ ] Comparative-fault language accurate — no "must be not at fault" framing (A.R.S. § 12-2505 pure comparative fault)
- [ ] No guarantees of results or outcomes
- [ ] Fee information (if included) specifies contingency structure and expense allocation per ER 1.5
- [ ] Aggregate substantiation figure: **$1.7B+** (canonical) — not "$150M+" unless scope-specific and documented

## 2. VISUAL REVIEW

- [ ] Crash footage contextualized (if used) — not gratuitous
- [ ] No gore/graphic injury imagery (Meta Personal Attributes + ER 7.1)
- [ ] Dramatizations / reenactments labeled
- [ ] Actors disclosed if non-clients appear
- [ ] AI-generated imagery disclosed as "Dramatization" where depicting "client" or "scene"
- [ ] No AI lip-sync deepfakes of real public figures (Meta ban)
- [ ] No AI-generated "testimonials" claiming case results

## 3. DISCLOSURES

- [ ] Responsible attorney (WCTL AZ-admitted attorney) name and contact info present
- [ ] "This is an advertisement" present on landing page
- [ ] "Fair Case is operated by Second Chair. Not a law firm." disclosure present (ER 7.5)
- [ ] TCPA consent language above submit button (FCC 2025 one-to-one consent, naming WCTL)
- [ ] TrustedForm + Jornaya integration active
- [ ] Multi-jurisdictional disclosure if WCTL's AZ licensing requires clarification
- [ ] Disclosures clear and conspicuous — matching size, color, contrast of primary content (ER 7.1)
- [ ] ER 1.18 prospective-client disclaimer above submit button

## 4. LEAD GENERATION COMPLIANCE

- [ ] Second Chair does NOT recommend WCTL (presents as neutral lead connector)
- [ ] No implication that Second Chair has analyzed the person's legal problem
- [ ] No implication that the referral is unpaid
- [ ] WCTL identified as the lawyer/firm responsible for the ad content
- [ ] Client informed-consent framework in place for ER 1.5(f) Second Chair compensation disclosure (in WCTL engagement letter)
- [ ] No selection language ("we'll match you" / "best fit for you" / "we recommend")

## 5. PLATFORM COMPLIANCE

- [ ] Special Ad Category settings correct (Meta, TikTok)
- [ ] No Personal Attributes policy violations (no "You" + injury/pain/condition)
- [ ] Platform-specific creative guidelines met
- [ ] Landing page matches ad claims (no bait-and-switch)
- [ ] No targeting based on PII-matched audiences of identified victims (ER 7.3 risk)

## 6. FORM ARCHITECTURE

- [ ] "Where did the crash happen?" question with tribal-land branching (see [`06_Tribal_Jurisdiction.md`](06_Tribal_Jurisdiction.md))
- [ ] "Was a government vehicle involved?" question for 180-day notice triage (see [`03_Statutes/12-821-01_Government_Claim_Notice.md`](03_Statutes/12-821-01_Government_Claim_Notice.md))
- [ ] Date-of-incident question with 2-year SOL filter + soft-decline branch (see [`03_Statutes/12-542_Statute_of_Limitations.md`](03_Statutes/12-542_Statute_of_Limitations.md))
- [ ] Soft-tissue gate matches WCTL underwriting criteria
- [ ] Form does NOT solicit case strategy, settlement views, or detailed legal facts (ER 1.18 information-firewall)
- [ ] Form does NOT include AI-driven case-strength scoring shown to claimant

## 7. SPANISH-LANGUAGE COMPLIANCE (If Spanish Creative Used)

- [ ] Professional translation with translator's certification on file (no machine translation alone)
- [ ] Disclosures translated faithfully (no softening of legal qualifications)
- [ ] Same prominence — Spanish disclosures match English in size, color, contrast
- [ ] Spanish landing pages tested with native speakers
- [ ] Spanish TCPA consent block in place
- [ ] Spanish ER 1.18 disclaimer in place
- [ ] Spanish ER 7.5 trade-name disclosure in place

## 8. SUBSTANTIATION

- [ ] Substantiation file current (see [`13_Substantiation_File_Spec.md`](13_Substantiation_File_Spec.md))
- [ ] Aggregate figure: $1.7B+ (canonical)
- [ ] Case-outcome rate (e.g., 97%) backed by closed-case data
- [ ] Specific case-result claims (e.g., $8.4M / $4.1M / $2.3M) backed by court filings + settlement docs
- [ ] Review aggregates (e.g., 4.9/5 from 254 reviews) backed by dated screenshots
- [ ] Years-of-experience claims backed by Bar admission dates

## 9. LEGAL ARCHITECTURE

- [ ] WCTL engagement letter for AZ matters includes:
  - [ ] Contingent fee percentages (settlement / trial / appeal) per ER 1.5(c)(1)
  - [ ] Treatment of expenses (before or after fee calculation) per ER 1.5(c)(2)
  - [ ] Client liability for expenses if not prevailing per ER 1.5(c)(3)
  - [ ] Disclosure of Second Chair compensation arrangement per ER 1.5(f)
  - [ ] Identification of Second Chair as recipient and the share
  - [ ] Non-interference acknowledgment per ER 1.5(f)(3)
  - [ ] Closing-statement promise (written outcome statement at conclusion) per ER 1.5(c)
- [ ] AZ-admitted attorney identified as responsible attorney of record
- [ ] AZ-admitted attorney's bar number documented
- [ ] WCTL letterhead/website discloses jurisdictional admissions per attorney (ER 7.1 transparency)

## 10. SUPPORTING DOCUMENTATION

- [ ] Practice 2.0 voluntary review submitted for landing page templates (recommended; not required)
- [ ] Practice 2.0 review log entry created if submitted (see [`14_Voluntary_Review_Practice_2.0.md`](14_Voluntary_Review_Practice_2.0.md))
- [ ] AZ-admitted attorney good-standing certificate current (annual)
- [ ] Defense counsel relationship in place for complaint response (see [`15_Disciplinary_Process_and_Sanctions.md`](15_Disciplinary_Process_and_Sanctions.md))
- [ ] WCTL substantiation file location confirmed: `Abracadabra/08_Brands/Second_Chair/23_Clients/West_Coast_Trial_Lawyers/_Research/wctl_substantiation.md`

## 11. DIRECT MAIL / EMAIL (If Applicable — SC Currently Doesn't Use These)

- [ ] Physical mail: "NOTICE: THIS IS AN ADVERTISEMENT!" in red ink, all caps, double-size font on envelope and first page
- [ ] Email: "Advertising Material" in subject line
- [ ] Email: "Advertising Material" at beginning and end of message, double-size font
- [ ] Accurate sender/routing information
- [ ] Opt-out mechanism functional

## 12. SIGN-OFFS

- [ ] WCTL AZ-admitted attorney sign-off (responsible attorney)
- [ ] WCTL compliance lawyer sign-off (if separate)
- [ ] SC compliance lead sign-off
- [ ] Davis sign-off (if scope warrants)

---

## How to Use This Checklist

1. **Before any new AZ creative variant launches**, run this checklist end-to-end
2. **Document the run** — date, asset ID, who reviewed
3. **If a box can't be ticked**, document the gap and the remediation plan; campaign waits
4. **Re-run quarterly** for any persistent campaigns to catch drift
5. **Re-run after any rule change** (AZ Supreme Court orders, new disciplinary precedent, FCC rule amendments)
